# 🌱 Искра и Росток — Генератор контента

Добро пожаловать в центр подготовки контента «Искра и Росток»! Этот проект содержит инструменты для генерации экспертных статей (в стиле Аналитик/Ботаник) на основе ваших баз знаний.

Процесс максимально прост и линеен. Никаких ботов, сложных планировщиков и скрытых API.

---

## 🚀 Как работает конвейер (Два простых шага)

### Шаг 1: Добавление знаний
Поместите ваш исходный `.md` файл (например, заметку из Obsidian) в папку `/knowledge`.

### Шаг 2: Генерация и Экспорт
Запустите основной скрипт. Он автоматически найдет новые файлы, сгенерирует по ним текст статьи и создаст удобное HTML-превью для просмотра.

```bash
python src/scheduler.py
```

### Результат:
1.  Готовый текстовый черновик статьи (Markdown) появится в папке `/drafts`.
2.  Стильный HTML-файл для чтения появится в папке `/drafts/preview`.

Вам остается только открыть HTML-файл в браузере, проверить текст, сгенерировать картинки по промптам из блока "Галерея" и вручную скопировать материал в Яндекс.Дзен или Telegram! Удачи!
