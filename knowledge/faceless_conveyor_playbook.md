Кейс 1: Faceless-конвейер коротких видео (YouTube Shorts / Instagram Reels) на зарубежную аудиторию

1. Архитектура автоматизации (Связка n8n + Gemini)

Алгоритм работы n8n (визуальный скрипт/workflow):

Шаг 1: Триггер (Cron Node / Schedule Trigger). Настраивается на запуск 1 раз в неделю (например, в воскресенье ночью).
Шаг 2: Чтение данных (Google Sheets Node). n8n подключается к таблице через OAuth2. Извлекает строку с темой видео (например, "Инженерные фейлы истории: Падение моста Такома") и отмечает ее как "В работе".
Шаг 3: Генерация контента (HTTP Request Node к API Gemini Advanced). Отправляется POST-запрос со строгим системным промптом.
  - Промпт: "Ты - эксперт по удержанию внимания в YouTube Shorts. Напиши сценарий на 45 секунд. Первые 2 секунды - жесткий хук. На 25-й секунде - кульминация. В конце - призыв подписаться. Верни ответ строго в JSON формате: { 'script': 'текст для озвучки', 'prompts': ['промпт 1', 'промпт 2', ...], 'tags': ['тег1', 'тег2'], 'description': 'описание' }."
Шаг 4: Парсинг JSON (JSON Parse Node). Разбивка ответа Gemini на 3 переменных: Сценарий, Промпты, Метаданные.
Шаг 5: Сохранение результатов (Google Drive Node / Local File System Node).
  - Создается папка с названием темы.
  - Сохраняется script.txt (для озвучки).
  - Сохраняется prompts.txt (для Flux).
  - Сохраняется meta.txt (Теги и описание).
Шаг 6: Обновление таблицы (Google Sheets Node). Статус темы меняется на "Сценарий готов".

2. Визуальный поток (Локальный Flux + CapCut PRO)

Шаг 1: Локальная генерация изображений (Flux на Radeon RX 6900 XT).
  - Использование графического интерфейса ComfyUI (он лучше оптимизирован для пакетной генерации) или Forge.
  - Настройка Workflow в ComfyUI: загрузка списка промптов из prompts.txt (через ноду Load Lines From Text File), подключение к чекпоинту Flux.1-schnell (он быстрее и потребляет меньше VRAM, 16GB VRAM RX 6900 XT хватит с запасом).
  - Запуск пакетной генерации (Batch Count = количество промптов). Изображения сохраняются в папку проекта. Соотношение сторон 9:16 (вертикальное).

Шаг 2: Сборка в CapCut PRO.
  - Импорт: Перетаскиваем все сгенерированные картинки на таймлайн.
  - Озвучка: Копируем текст из script.txt. В CapCut выбираем инструмент Text-to-Speech. Выбираем премиальный ИИ-голос (на английском). CapCut сгенерирует аудиодорожку.
  - Синхронизация: Растягиваем/обрезаем картинки под ритм озвучки.
  - Субтитры: Выбираем "Auto Captions" -> включаем динамические субтитры с выделением слов. Применяем премиум-шаблоны анимации текста.
  - Эффекты: Добавляем переходы (Zoom, Pan) для динамики статичных картинок.
  - Экспорт: Вывод в 1080x1920 (4K не нужен для Shorts), 60 FPS для плавности.

3. Финансовый шлюз и Безопасность в РФ (Вывод денег)

Схема 1: Вывод через СНГ-карты (AdSense на другое ГЕО)
  - Механика: Покупка удаленного оформления карты в Киргизии или Казахстане (ИИН + Карта). Регистрация почты через VPN, покупка номера SMS-активации той же страны, регистрация AdSense на этот номер и страну. Получение PIN-кода AdSense на арендованный адрес в той стране.
  - Плюсы: Прямая работа с Google, 100% дохода.
  - Минусы: Долго (доставка PIN-кода 3-4 недели). Риск блокировки AdSense, если Google заподозрит несоответствие ГЕО, VPN и реального IP. Налоги нужно платить/декларировать самостоятельно.

Схема 2: Работа через Медиасети (MCN)
  - Механика: Подключение канала (когда выполнены условия монетизации - 1000 подписчиков, 10 млн просмотров Shorts) к партнерской сети (например, Yoola, Mediacube).
  - Плюсы: MCN выступает гарантом. Вывод денег возможен на российские карты, крипту (USDT) или электронные кошельки. Нет проблем с AdSense (канал привязывается к AdSense сети). Меньше рисков бана за ГЕО.
  - Минусы: Комиссия сети (обычно 15-20% от дохода). Требуют пройти строгую модерацию контента.

Вердикт для старта с нуля:
Безопаснее и надежнее для новичка - Схема 2 (Медиасети MCN).
Обоснование: Привязка иностранного AdSense через виртуальные адреса и VPN требует опыта арбитража трафика и сопряжена с высоким риском безвозвратного бана аккаунта с заморозкой средств. MCN берет комиссию за избавление вас от юридических, налоговых и санкционных рисков. Для Faceless-канала на первом этапе важнее стабильность вывода (в USDT), чем потеря 15-20% комиссии.
